Description

Abstract class for defining P/T-Nets, which are a special kind of Petri nets that only consider one token type (tokens are indistinguishable).<br> <br> The class defines specific markings that respect P/T-Net properties.<br> Markings contain for each place the number of tokens it contains. By providing appropriate methods, the class allows to add relations having weights<br> (which define the number of tokens transferred over the relation). @author Thomas Stocker
