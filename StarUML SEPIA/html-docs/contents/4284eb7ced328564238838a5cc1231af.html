Description

Abstract class that defines general properties for places of place/transition nets (P/T Nets).<br> This class inherits basic Petri net place properties from {@link AbstractPlace}.<br> It is parameterized with a specialized type of relations which can be added ({@link AbstractPTFlowRelation}). <br> P/T Nets consider only one token type.<br> The state of a P/T Net place is determined by the number of tokens it holds,<br> thus the state type is set to {@link Integer}. @author Thomas Stocker @param <E> The type of flow relations connected to the place.
