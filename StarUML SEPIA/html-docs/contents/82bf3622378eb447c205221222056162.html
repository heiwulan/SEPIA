addDeclassificationTransition

UMLOperation
Untitled :: JavaReverse :: de :: uni :: freiburg :: iig :: telematik :: sepia :: petrinet :: ifnet :: abstr :: AbstractIFNet :: addDeclassificationTransition

Description

Adds a declassification transition with the given name to the IF-Net.<br> Transitions names have to be unique. In case the net already contains a transition with the given name, no transition is added to the net. @param transitionName The name for the declassification transition. @return <code>true</code> if the transition was successfully added to the net;<br> <code>false</code> otherwise. @throws ParameterException If the transition name is <code>null</code>.
