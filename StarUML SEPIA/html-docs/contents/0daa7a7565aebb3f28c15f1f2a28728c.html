Description

Analysis context of IF-Nets.<br> Analysis contexts contain information about: <ul> <li>Activity classification: The security level of process activities (IF-Net transitions).</li> <li>Attribute classification: The security level of process attributes (Colored tokens of the IF-Net).</li> <li>Subject clearance: The clearance level of subjects executing process activities.</li> <li>Subject descriptors: Subjects assigned to process activities.</li> </ul>
